# 当前 MVP 项目说明书

## 项目定位

本项目当前方向是一款大战略多周目养成卡牌 RPG。核心参考《兰斯10 决战》中的多战线取舍、角色卡资产、势力战力和多周目收集思路，但只借鉴结构体验，不复刻原作角色、剧情、名称、数据或世界观。

玩家面对的是战争进行时的王道幻想世界，而不是灾后清扫、单人修仙或传统手牌对战。MVP 只截取更大战争中的一个局部：人类城堡势力与高等精灵森林势力在同一战区中合作、竞争、撤退与牺牲。

## 核心体验

第一阶段关注有限回合、有限行动和多战线压力。玩家每轮都要在推进主线、救援角色、稳住势力战力、获取新卡和规避损失之间取舍。设计目标不是让玩家一次完成所有目标，而是制造无法全救、路线错过、角色死亡或离队的结果，并把这些结果沉淀为多周目复盘与收集动力。

角色卡是长期资产，不是传统卡牌游戏中的手牌。玩家收集的是角色的不同职业版本卡；这些卡可以成长，归属于某个势力，并贡献势力整体 HP/AT。角色作为领队或上场版本时，才提供该版本对应的主动技能、被动技能或战场定位。

## MVP 卡池

MVP 只包含 2 个势力、6 名女性角色、12 张角色卡。每个势力 3 名角色；每名角色 2 个版本：正常版与泳装类职业版。

泳装版不是纯皮肤，而是同一角色的另一张职业卡或版本卡。衣装可以更轻、更露肤，但必须保留职业识别与阵营识别，避免变成与玩法无关的换装图。

## MVP 势力

### 城堡阵营

城堡阵营是人类城堡势力，整体气质接近传统王道幻想中的城堡、人类、骑士与圣堂军队。视觉以金色、象牙白、银色为主，职业方向包括骑士、魔法师、牧师等。参考方向只限于阵营气质和职业轮廓，不使用其他作品的专有设定。

- 城堡 A：18 岁，162cm，45kg，B cup，女骑士。
- 城堡 B：18 岁，150cm，38kg，A cup，魔法师。
- 城堡 C：20 岁，166cm，53kg，E cup，女牧师。

### 森林阵营

森林阵营是高等精灵势力，设定为白色皮肤、长寿、纤细、长耳朵的森林文明。视觉以绿色、浅玉色、叶绿为主，并用金叶点缀表现高等精灵的古老与精致。

- 精灵 A：180cm，55kg，精灵女战士。
- 精灵 B：175cm，50kg，精灵弓箭手。
- 精灵 C：170cm，47kg，精灵祭祀。

## 角色版本与势力战力

每名角色同一周目只能选择其中一张版本卡作为上场或领队版本。例如城堡 A 的正常女骑士版与泳装骑士版不能同时作为两个单位出战。

已获得的同角色多个版本卡仍然都是完整资产。只要角色没有死亡或离队，这些版本卡都会贡献所属势力的 HP/AT，形成类似多卡资产累积的势力战力。也就是说，出战选择决定战场技能与定位；收集深度决定势力底盘。

如果角色在本周目死亡或离队，该角色的图鉴、多周目记录和历史获得情况保留，但本周目不可出战，也不贡献所属势力战力。死亡和离队应成为路线取舍的一部分，而不是单纯的数值惩罚。

## 当前美术方向

MVP 卡牌美术参考用户提供的柔和动画感画风：软光、水彩感、薄线条、漂浮光粒和空气感。城堡阵营偏金白、银白和圣堂质感；森林阵营偏绿、浅玉、叶绿和金叶装饰。

角色设计需要减少约 40% 纹饰，扩大干净色块，让轮廓和职业标志更容易阅读。每个职业只保留 2 到 3 个关键标志物，例如骑士的轻甲与剑、魔法师的法杖与法袍、牧师的圣徽与治疗道具、精灵弓箭手的长弓与叶形饰件。角色姿势需要避免同质化，让职业、性格和版本差异在第一眼能被区分。

## 第一阶段暂不展开

本说明书只作为当前 MVP 事实来源，不展开完整系统设计。第一阶段暂不定义完整战斗公式、完整世界观、全部结局、完整技能表和完整 UI。后续 Symphony issue 应以本文的项目定位、卡池范围、势力设定、版本规则、势力战力规则和美术方向为准继续拆分。
