# Project Overview

`rance` is a data-driven 2D RPG card battle prototype. The current repository
contains planning notes and minimal content fixtures only; runtime gameplay,
UI, scenes, and build automation are intentionally out of scope for this issue.

## Current Scope

- Define the first content data shape under `data/`.
- Provide small JSON fixture collections for cards, characters, enemies,
  encounters, quests, rewards, and progression.
- Document conventions that later Godot tasks can consume without changing the
  content IDs.
