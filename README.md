# rance

Godot 4.x project skeleton for a long-form grand strategy card RPG.

- Open the project through [project.godot](project.godot).
- Start implementation work from the canonical docs hub:
  [docs/README.md](docs/README.md).
- Current project direction is summarized in
  [docs/project-overview.md](docs/project-overview.md).
- Module boundaries are indexed in [docs/modules/README.md](docs/modules/README.md).
- Content data schemas and fixtures are documented in [data/README.md](data/README.md).
- Historical request notes, not current source of truth, are preserved under
  [docs/requests/](docs/requests/README.md).

This skeleton intentionally contains no gameplay, card, combat, strategy, story,
or balance implementation.
